#  Customer Sales & Profitability Dashboard (Excel Project)

##  Project Overview
This project presents a dynamic and interactive Excel dashboard designed to analyze customer sales performance and profitability.
The dashboard transforms raw transactional data into meaningful business insights using advanced Excel features such as Pivot Tables, KPI cards, conditional formatting, and interactive filters.
It is built to simulate a real-world business scenario where decision-makers need clear visibility into revenue, profit margins, customer performance, and loss-making orders.

##  Business Objectives

* Analyze overall sales and profitability performance
* Identify loss-making transactions
* Track monthly sales trends
* Compare performance across product categories
* Identify top-performing customers
* Calculate and evaluate profit margins

##  Tools & Excel Features Used

* Excel Tables (Structured References)
* Pivot Tables
* Pivot Charts
* Slicers for interactive filtering
* IF Function
* SUM, SUMIF, COUNTIF
* Profit Margin Calculation
* Conditional Formatting
* Sorting & Filtering
* KPI Card Design
* Dashboard Layout & Design Principles

##  Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Profit Margin (%)
* Total Orders
* Loss Orders Count
* Monthly Sales Trend
* Category-wise Performance
* Top Customers by Revenue

##  Dashboard Highlights

✔ Clean and professional dashboard layout
✔ Interactive filtering using slicers
✔ Clear visualization of profit trends
✔ Identification of low-margin / loss-making transactions
✔ Category-wise and customer-wise performance comparison

##  Business Insights Generated
* Certain product categories contribute higher profit margins.
* Some transactions generate losses and impact overall profitability.
* Sales performance varies month-to-month, indicating seasonal patterns.
* A small group of customers contributes significantly to total revenue.
